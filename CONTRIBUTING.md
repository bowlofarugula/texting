# Contributing

Contributions are welcome — thanks for your interest in improving this project!

- **Issues** — bug reports, ideas, and questions are all welcome. For **security** issues,
  please report privately rather than opening a public issue (see `SECURITY.md` if present).
- **Pull requests** — encouraged. For anything substantial, open an issue first so we can
  agree on the approach. Keep changes focused, match the existing style, run the project's
  checks (see the README), and add tests where it makes sense.
- **Dev setup** — see the README for how to run it locally.

By contributing, you agree your contributions are licensed under this project's Apache-2.0 license.
